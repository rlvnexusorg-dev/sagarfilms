# **App Name**: Sagar Films Studio

## Core Features:

- Booking Calendar: Display availability and allow clients to book photography or videography sessions for weddings, pre-wedding events, Haldi ceremonies, and birthdays.
- Package Selection: Offer customizable packages with options for different event types, durations, and deliverables (e.g., edited photos, video highlights).
- Portfolio Showcase: Display high-quality photos and videos from previous events to showcase the studio's style and capabilities. Use AI tool to categorize portfolio entries using content analysis.
- Client Testimonials: Feature client testimonials and reviews to build trust and credibility.
- Contact Form: Allow potential clients to submit inquiries and contact the studio directly.
- Automated Invoice Generation: Automatically generate invoices based on selected packages and bookings.
- Location and Contact Info: Display studio location, phone number, and email address prominently.

## Style Guidelines:

- Primary color: Deep burgundy (#800020) for elegance and sophistication.
- Background color: Off-white (#F5F5DC), a very slightly tinted version of the primary color, but far less saturated, for a soft, classic feel.
- Accent color: Muted gold (#B8860B), an analogous color shifted in hue, for subtle highlights and CTAs.
- Headline font: 'Playfair', a modern serif font, for titles.
- Body font: 'PT Sans', a humanist sans-serif, for text and form elements.
- Use clean, minimalist icons related to photography, videography, and event types.
- Emphasize large, high-quality images and videos in a grid or slideshow format.
- Subtle, tasteful animations for loading transitions and image reveals.
